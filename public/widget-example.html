Пример использования виджета каталога

Инструкция по использованию

Виджет каталога инициализируется в контейнере ниже. Для использования на вашем сайте:

  1. Подключите React и ReactDOM из CDN (или используйте ваши версии)
  2. Подключите CSS файл виджета: widget-catalog.css
  3. Подключите JS файл виджета: widget-catalog.js
  4. Создайте контейнер для виджета: <div id="widget-catalog"></div>
  5. Инициализируйте виджет в скрипте
// Пример кода инициализации: function initWidgetCatalog() { const widget = new window.WidgetCatalog({ el: "#widget-catalog", dealers: [], // Массив ID дилеров, пустой массив = все дилеры }); widget.run(); }
